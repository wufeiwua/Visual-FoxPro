*题目：编程求P=1-1/(2×2)+1/(3×3)-1/(4×4)+1/(5×5)。
*      将结果存入变量OUT中，要求用Do While语句实现。

*-------------------------------------------------------
SET TALK ON
SET LOGERRORS ON
OUT=-1
**********Program**********



**********  End  **********

SET LOGERRORS OFF
SET TALK OFF
答案：----------------------
P=0
M=1
I=0
DO WHILE M<=5
P=P+((-1)^(M+1))/(M*M)
M=M+1
ENDDO
? "P=",P
OUT=P
